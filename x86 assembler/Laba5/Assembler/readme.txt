Warning!!! файл должен иметь расишрение asm.
ctrl + f4 - refresh mounted drive
dir - показать содержимое текущей папки
exit - выйти из программы
с.bat "filename" - компиляция (*.asm => *.obj получить объектный файл !!! Обязательно !!!)
l.bat "filename" - линковка (*.obj => -.exe !!! Обязательно !!!)
d.bat "filename" - отладка
all.bat "filename" - компиляция, линковка и отладка одной командой
Меню отладчика
view -> register - открыть окно с содержимым регистров все работает в 16-й СС
( https://calcus.ru/perevod-sistem-schisleniya/iz-shestnadcaterichnoy-v-desyatichnuyu для особо глупых )
жмякать f8 для следующего шага
file -> quit - завершить работу программы
